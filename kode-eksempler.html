Kode Eksempler

På denne siden viser vi frem noen praktiske eksempler på hvordan du kan implementere bærekraftige programmeringspraksiser i koden din. Eksemplene dekker områder som effektiv løkking, databehandling og hvordan du kan bruke teknikker som **lazy loading** for å gjøre nettsider mer energieffektive. Målet er å gi deg konkrete verktøy som du kan bruke for å redusere ressursforbruket i programvaren din, og på den måten bidra til en mer bærekraftig teknologiutvikling.

Effektiv løkking

Unngå unødvendige operasjoner i løkker for å spare prosessorkraft. Her er et eksempel på hvordan du kan forbedre løkken for å gjøre koden mer effektiv:


Eksempel
names = ["Anna", "Bob", "Cecilie", "David"]
for i in range(len(names)):
  print(names[i])

Grønnere kode
names = ["Anna", "Bob", "Cecilie", "David"]
for name in names:
  print(name)

Databehandling

Jobb med mindre datamengder der det er mulig for å bruke mindre minne. Ved å behandle dataene sekvensielt i stedet for å laste hele datasettet på en gang, kan du redusere minnebruken og forbedre ytelsen:


Eksempel
data = load_big_dataset()
filtered_data = [item for item in data if item.is_relevant()]
process(filtered_data)

Grønnere kode
for item in stream_dataset():
  if item.is_relevant():
    process(item)
